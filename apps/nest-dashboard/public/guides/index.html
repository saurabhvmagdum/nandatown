Pick how you want to work.
Step 1/10
terminal the app where you type commands (Terminal on Mac, PowerShell on Windows)
repo a project folder on GitHub (here, projnanda/nandatown)
fork your own copy of the project to work on
clone downloading that copy onto your computer
branch a safe side version (name yours hackathon/ plus what it does)
commit a saved checkpoint with a note that says what changed
plugin your own version of one of the 12 layers
pull request asking the project to add your changes
Nothing joins the real project until a maintainer accepts your PR, so you cannot break anything by trying.
